Info

Wacom GUI is a Python 3.7+/PyQt5-based user interface for setting xsetwacom commands for tablets. It has been developed primarily on the MATE desktop, however development has moved to gnome-classic. It should work for others as well. If you have any issues/feature requests, please submit them to the project git page: Wacom GUI Additionally, I have had assistance in continued development from many others, without which this project wouldn't be where it is. Stylus

This page allows you to:
  • Alter pressure sensitivity curve
  • Switch pen detection mode between Absolute and Relative
  • Swap side button inputs on the stylus
  • Switch between 'hover' and 'touch pad' mode for input detection (Pen Touch)

Eraser

This page allows you to:
  • Alter pressure sensitivity curve
  • Switch pen detection mode between Absolute and Relative

Pad

Will only appear of the tablet has pad buttons available.
  • Assign key press(es) to Pad Button
    • Click the button, press the key command you wish to record, then click the button again to save.
  • Disable Pad Button
    • Click the green circle next to the button, it will become disabled.
  • Reset All Pad Buttons
    • Click the reset button to return all buttons to their default state.
  • Disable/Enable All Pad Buttons
    • Click the enable/disable button to toggle all button's state.

Touch

Will only appear if tablet allows for touch input.
  • Enable/disable Finger Touch capability

Cursor

Will only appear if tablet has a cursor input.
  • Switch cursor detection mode between Absolute and Relative

Other Settings

  • Map tablet input to a specific monitor or all (if more than one)
  • Switch tablet between left and right orientation
  • Restore tablet to all default settings

General Notes

  • By default, pressing "Win+z" keys together will toggle the tablet mapping between all the monitors then to all monitors. This can be changed via "Keyboard Shortcuts".
  • Config files are stored and loaded from ~/.wacom-gui/<deviceid>/default.sh
  • Currently only one device can be hooked up at a time to configure; multiple devices of the same type will currently not work.
